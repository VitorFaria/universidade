# universidade

Pequeno CRUD simulando cadastros em uma universidade.

Para rodar o backend, importe-o no Eclipse e rode como um spring boot app após atualizar com o Maven e atualizar o arquivo de configurações de banco (application.properties). Após o spring criar as tabelas necessárias, rode o inserts.sql para preencher o banco com dados de teste.

Para rodar o frontend, vá para o diretório e rode "npm install" seguido de "ng serve" no terminal.

A url da tela inicial é http://localhost:4200. Logue com usuário e senha "alex123".
